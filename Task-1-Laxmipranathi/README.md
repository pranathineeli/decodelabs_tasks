# NovaTech - Frontend Engineering Landing Page

## Project Overview

NovaTech is a modern responsive static landing page built using semantic HTML5 and CSS3.  
The project focuses on clean frontend architecture, accessibility, responsive layouts, and scalable CSS engineering principles.

---

## Features

- Semantic HTML5 structure
- Responsive layout design
- CSS Grid and Flexbox implementation
- Accessibility-focused development
- Modern UI components
- Reusable CSS architecture
- Smooth hover interactions
- Performance-optimized image handling
